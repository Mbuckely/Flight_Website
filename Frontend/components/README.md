# Reusable Components

Reusable UI lives here, grouped by feature or layout area.

| Folder | What Goes Here |
| --- | --- |
| `components/layout` | Site-wide layout pieces like `Navbar`, `Footer`, and `Sidebar`. |
| `components/home` | Home page sections like `Hero`, `Features`, and dashboard previews. |
| `components/flights` | Flight search and result components like `AirportField`, `SearchBar`, and `FlightCard`. |
| `components/places` | Place/location inputs such as `GooglePlaceField`. |
| `components/ui` | Small generic UI helpers like reveal/animation wrappers. |

If a component is only used by one page and grows large, keep it near that feature folder. If it becomes useful in multiple places, move it here.
